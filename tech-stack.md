# Technology Stack

## Frontend
- React with TypeScript
- Tailwind CSS for styling
- React Router for client-side routing

## Backend
- Node.js with Express and TypeScript
- Prisma ORM
- PostgreSQL database

## Authentication
- Database sessions (session tokens stored in DB, not JWTs)

## AI
- Claude API (Anthropic) — classification, summaries, suggested replies

## Email
- SendGrid or Mailgun for inbound email ingestion and outbound sending (TBD)

## Deployment
- Docker for containerisation
- Cloud provider TBD (e.g. AWS, GCP, or Azure)
